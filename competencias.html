Perfil Competências Cursos Contato

Principais competências

React.js:

Desenvolvimento de interfaces dinâmicas e responsivas, utilizando hooks, context API, e integração com APIs RESTful.

Node.js:

Criação de APIs RESTful e microserviços com Node.js e Express.js. Implementação de autenticação e autorização de usuários com JWT e OAuth. Integração com bancos de dados SQL (SQLServer, PostgreSQL)

React Native:

Criação de aplicativos móveis nativos para iOS e Android, com integração de funcionalidades nativas, como câmera, GPS e armazenamento local. Publicação de aplicativos nas Google Play Store e Apple App Store.

Coletores de Dados (RFID e Código de Barras):

Integração e programação de coletores de dados RFID e coletores de código de barras. Desenvolvimento de soluções para captura e processamento de dados em tempo real. Implementação de sistemas de rastreamento e controle de inventário.

HTML5:

Estruturação de páginas web com foco em acessibilidade e semântica.

Javascript / Typescript:

Sólidos conhecimentos em ES6+, manipulação de DOM, eventos, e async/await. Aplicação de tipagem estática, interfaces e classes para melhorar a manutenção e robustez do código, com foco na segurança e prevenção de erros

CSS3 / Tailwind CSS :

Estilização de interfaces com ênfase em layouts responsivos, animações e transições e Tailwind CSS para desenvolvimento rápido e eficiente de layouts personalizados.

Versionamento de Código:

Experiência com Git para controle de versão e colaboração em equipe.

Ferramentas de Build:

Uso de Webpack, Babel e outros para otimização de código e bundling.
